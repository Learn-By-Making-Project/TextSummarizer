# 📄 Text Summarizer

## 📌 Project Description
This project is a text summarization tool that helps users quickly generate summaries from long text inputs using modern techniques.

---

## ⚙️ Installation

1. Clone the repository:
   git clone https://github.com/your-username/TextSummarizer.git

2. Navigate to the project folder:
   cd TextSummarizer

3. Install dependencies:
   npm install

4. Run the project:
   npm start

---

## 🚀 Usage
Provide input text and get a summarized output instantly.
