# AI Text Summarizer — Gemini CLI Setup Instructions

## ⚠️ Important Notice

* ❌ DO NOT write any actual code
* ❌ DO NOT implement features
* ✅ ONLY create folder structure and empty files
* ✅ Follow instructions carefully

This repository is for **learning collaboration, workflow, and reading documentation**, not coding immediately.

---

## 📌 Objective

Set up the initial project structure using CLI tools so contributors can:

* Understand project layout
* Pick issues
* Start implementing features themselves

---

## 🛠️ Setup Instructions (Using Gemini CLI)

### Step 1: Create Project Folder

```bash
mkdir text-summarizer
cd text-summarizer
```

---

### Step 2: Initialize Git Repository

```bash
git init
```

---

### Step 3: Create Folder Structure

```bash
mkdir frontend backend docs
```

---

### Step 4: Create Frontend Files (Empty)

```bash
cd frontend
touch index.html style.css script.js
cd ..
```

---

### Step 5: Create Backend Files (Empty)

```bash
cd backend
touch app.js
mkdir routes
cd ..
```

---

### Step 6: Create Documentation Files

```bash
cd docs
touch workflow.md
cd ..
```

---

### Step 7: Create Root Files

```bash
touch README.md .gitignore
```

---


### 2. Commit


### 3. Push to GitHub

